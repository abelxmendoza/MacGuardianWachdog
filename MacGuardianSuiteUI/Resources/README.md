# Logo Assets

## Adding the MacGuardian Logo

To add the MacGuardian Watchdog logo to the app:

1. **Save your logo image** in this directory (`MacGuardianSuiteUI/Resources/`)
2. **Name it exactly**: `MacGuardianLogo.png` (or `.jpg`/`.jpeg`)
3. **Recommended size**: 512x512 pixels or higher for best quality
4. **Format**: PNG with transparency preferred, or JPG

The logo will automatically appear in:
- Welcome screen
- Dashboard (optional - can be added)
- App icon (if configured)

## Supported Formats
- PNG (recommended)
- JPG/JPEG
- SVG (if converted to PNG)
